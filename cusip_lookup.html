CUSIP/Ticker Lookup & Moneydance Import Tool

Input Format

Each line should have: TICKER/CUSIP followed by security name

AAPL Apple Inc
018522300 ALLETE INC
MSFT Microsoft Corporation
GOOGL Alphabet Inc

Step 1: Paste Your Securities List


Workflow Summary

  1. Paste securities with format: TICKER/CUSIP Name
  2. Click "Prepare Lookup Table" to parse the data
  3. Use search links to fill in missing CUSIP or Ticker information
  4. Download the Moneydance script and run it: Extensions → Python → Run Script

Script Configuration:

  • CREATE_NEW_SECURITIES = True - Create securities that don't exist
  • UPDATE_EXISTING_NAMES = False - Update names of existing securities
  • DRY_RUN = False - Preview changes without saving
